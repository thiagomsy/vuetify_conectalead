# ConectaLead - Vuetify

## ⚙ Técnologias usadas
    ✔️ Node v16.14.2
    ✔️ Vue v2.6.14
    ✔️ Vuetify v2.6.0

## 👥 Clonar o repositório
```
git clone https://github.com/thiagomsy/vuetify_conectalead.git
```

## ➡️ Inicializar o Projeto
```
cd vuetify_conectalead
```
```
npm install
```

## 🚀 Como rodar o Projeto
```
npm run serve
```
